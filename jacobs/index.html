ABOUT COMPUTER LANGUAGES
Entire thought process of a computer: 1) fetch next instruction. 2) execute instruction. 3) go to 1).
They do this very very fast (like, billions of times a second - think about a billion for a minute).
"instructions" are just another number to the computer. They tell it to do things like "add these two numbers together" or "move this number from here to here".
("numbers" are usually 64-bits, but can be operated on in smaller slices: 32-, 16-, or 8-bits)
Java is a language that gets translated ("compiled") into these instructions. Java and other languages make it easier to write more complex and powerful programs.
